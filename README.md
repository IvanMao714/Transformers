## 运行环境

- python=3.6
- torch=1.10.1 (cpu)
- transformers=4.18.0
- datasets=2.4.0

## 安装

命令行下载对应的库

```terminal
pip install transformers
```

验证是否安装成功

```python
import transformers
```
