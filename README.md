# Huggingface Transformers工具教程

Transformer 越来越多出现在顶会论文中，其学习的必要性越来越重要

> 教程内容来源于官网教程：https://huggingface.co/course

## Contents

- 运行环境
- 安装
- 教程内容

## 运行环境

- python=3.6
- torch=1.10.1 (cpu)
- transformers=4.18.0
- datasets=2.4.0

## 安装

命令行下载对应的库

```terminal
pip install transformers
```

验证是否安装成功

```python
import transformers
```
